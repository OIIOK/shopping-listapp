# 🛒 쇼핑 리스트 앱

아이템 추가·삭제·체크 기능이 있는 순수 Vanilla HTML/CSS/JS 쇼핑 리스트 웹 앱입니다.

## 기능

- ✅ 아이템 추가 (버튼 클릭 또는 Enter 키)
- ✅ 아이템 삭제 (개별 삭제 / 완료 항목 일괄 삭제)
- ✅ 체크(구매 완료) 토글
- ✅ 필터: 전체 / 미구매 / 구매완료
- ✅ 통계 표시 (총 N개 중 M개 완료)
- ✅ localStorage로 데이터 영속 저장 (새로고침 후에도 유지)

## 사용법

`shopping-list.html` 파일을 브라우저로 열면 바로 실행됩니다. 별도 설치 불필요.

## 기술 스택

- HTML5
- CSS3 (Flexbox, `accent-color`, `:focus-visible`)
- Vanilla JavaScript (ES6+, localStorage API)
